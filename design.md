# AI 트렌드 2025 — 슬라이드 디자인 개선 가이드

## 현재 슬라이드 분석

### 구조
- 1슬라이드, 16:9 (10" × 5.625")
- 상단 헤더 + 2×2 카드 그리드 + 하단 푸터
- 다크 네이비 배경 (`0B1120`)

### 현재 색상 팔레트
| 역할 | 색상 | 용도 |
|------|------|------|
| 배경 | `0B1120` | 슬라이드 전체 |
| 헤더 | `131F35` | 상단 바 |
| 카드 | `1E2D45` | 4개 카드 배경 |
| 액센트 | `06B6D4` | 상단 라인·배지 |
| 텍스트 | `FFFFFF` / `94A3B8` | 제목 / 설명 |
